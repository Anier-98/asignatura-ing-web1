## Apuntes de la clase de HTML
la estructura de un sitio web es:
```<!doctype html>
<html>
    <head>
        <title>Mi primera pagina web</title>
    </head>
    <body>
        <h1>Bienvenido a mi pagina web</h1>
    </body>
</html>
```

Los lenguajes para la información estática son:
* HTML
* CSS
* JavaScript
